# @rimbu/reactor

This package offers integration of `@rimbu/actor` in React.

This package is still in an experimental stage.

For complete documentation please visit the _[Rimbu Docs](https://rimbu.org)_.

## Installation

To install:

> `yarn add @rimbu/reactor`

or

> `npm i @rimbu/reactor`
